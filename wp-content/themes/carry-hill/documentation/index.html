Theme Update - top

Step 1

It’s recommended that you make a backup of the theme prior to updating. Read this

Step 2

If you want to avoid having your site temporarily unsyled during the theme update you can install "Coming Soon" plugin. You can use somthing like this: plugin 1 plugin 2

Step 3

Note: If you have activated the child theme leave it as is. You only want to delete/update the theme.

From your WordPress dashboard, navigate to Themes. Activate a different theme, choose delete files for your theme, then upload the new theme and activate.

The same goes for following plugins: Bricklayer - Content Builder, Carry Hill Plugin, LayerSLider. You can freely deactivate them, delete them and reinstall them again with the theme, just like you would on the first theme install.

Additional Steps

If your server is slow and you are not able to complete the installation process please use FTP and replace the theme and the plugins directly.

It seems scary but you really won’t loose any of your data. The only exception is if you have made edits to theme files like header.php, footer.php etc. These changes will need to be made again to the new files. But all of your content and theme settings are preserved.

If you have already imported XML content and don't want to do that again. Here are Bricklayer shortcodes for the new Home page. Just create a new page and paste this code in the editor and press "Publish". Now you will have all Bricklayer bricks (widgets) available to you to see what are the settings if you wish to use something from this updated Home page.

Basic Installation - top

To install this theme you must have a working version of WordPress already installed. For information in regard to installing the WordPress platform, please see the WordPress Codex - http://codex.wordpress.org/Installing_WordPress

Extract the zipped package downloaded from ThemeForest to your desktop, in the extracted package you will find carry-hill.zip file which is the wordpress theme.

You can install the theme in two ways:

  • FTP: Extract carry-hill.zip file and upload the extracted folder to /wp-content/themes/ folder on your server.

  • WordPress: Navigate to Appearance -> Add New Themes -> Upload page. Select carry-hill.zip file. Press the Install Now button to upload and install the theme.

After uploading the theme, you have to activate it. Navigate to Appearence -> Themes page to activate the theme.

Installing and Activating Plugins - top

After theme installation you need to install and activate plugins this theme requires to work properly. You will see a list of required and recommended plugins. First you need to install them and than make sure you activate them.

Importing The Demo Data - top

We have included the exported XML file from our demo. (Please note that images from our demo site are not included.)

1.The XML file is included within the download package which will be used to import the content.

2. Navigate to WP Admin -> Tools -> Import.

3. Select and install WordPress importer.

4. Select the XML file and upload it. In a few minutes it will install and upload all content.

5. Now navigate to WP Admin -> Appearance -> Menus page and select the newly created menu and assign it to "Main Navigation" location.

6. Next, navigate to WP Admin -> Settings -> Reading Settings and set a front page.

Reading Settings - top

To setup one of these pages as Homepage, navigate to WP Admin -> Settings -> Reading Settings and set the newly created page as front page.

Posts page must be left blank.

Permalink Structure - top

Don't forget to set your permalink structure to have pretty links. This is important also for Breadcrumbs to be able to print full paths. So go to Admin/Settings/Permalinks and select something, recommended are "Day and name" and "Month and name". Click "Save Changes".

Menus - top

This theme supports custom WordPress menu with two level dropdown support.

To setup the menus, navigate to Apperance -> Menus page. Enter a name for your menu and add links to the menu by using the options given on the left sidebar. You can manage your menus using the drag and drop functionality. After setting up your menu, click on Manage Locations tab and assign your menu to a location on you site and hit the "Save" button. This theme has 2 menu locations Primary Menu and Secondary Menu, First is in the header and the second is in the footer.

Setting Up Blog Page - top

This page is a regular page that uses Bricklayer Content Builder Plugin, that is, it uses Post List Widget that lets you choose a category of posts to be displayed on a page.

If you have any trouble with displaying Blog page after importing XML data, just go to Blog page and scroll down so you see Post List Widget. Click the button in the right corner of the widget to edit it. Just select the category and press save on the widget and then save the page.

Events - top

We recommend that you install Events Made Easy plugin for managing events. We made some customizations to make this plugin play along with the theme more smoothly. To make your single event pages look like the demo you need to paste this code to Admin/Events/Settings/Events/Default single event format: 

        <p>#_EVENTIMAGE</p>
        <h4>#j #M #Y - #H:#i</h4>
        <h4>#_TOWN</h4>
        <p>#_NOTES</p>
        <p>#_ADDBOOKINGFORM</p>
        <p>#_MAP</p>

For event list widget:

List header format:
        <div class="ch-event-list">
List item format:
        <div class="ch-event-list-item"><h2 class="ch-border-title">#_LINKEDNAME</h2><div class="ch-event-date">#j #M #Y</div><div class="ch-event-location ch-borderless">#_TOWN</div></div>
List footer format:
        </div>

  2. Styling
  3. Additional

Shortcodes - top

The shortcodes are available under the usual formatting options on the post / page text editor. Make sure you are on the "visual editor" tab, the shortcode buttons are on the third row. Simply hover over a button to see which shortcode it is and then click it and the shortcode will be automatically added, then you need to populate the content.

Post Thumbnails - top

This theme uses the WordPress Post Thumbnails function, this way you can upload one image and WordPress will automatically resize it to all the different thumbnail sizes.

To set a post thumbnail for your post, go to Posts > Add New > Set Featured Image > the normal upload box will now appear, upload your image and click "Use as featured image".

How to Translate - top

The translation file is located in languages folder (/wp-content/themes/carry-hill/languages). Edit the .po file using POEdit, use the translation field to make replacements.

From the file menu, save file with your language name e.g de_DE.po. It will generate both a .po and .mo file for your translation.

Next edit wp-config.php located in the root folder of wordpress and define the WPLANG prefix with your language name which should be similar to the translation file name.
